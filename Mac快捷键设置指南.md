## 使用 **Automator 创建快速操作 + 快捷键**

这是最常见的方式，可以让你用快捷键直接启动任意程序。

1. 打开 **Automator**（在 `应用程序 → Automator`）。
2. 新建文稿 → 选择 **快速操作**。
3. 在左侧搜索框输入 **“启动应用程序”**（或 Launch Application）。
4. 把“启动应用程序”拖到右侧工作区。
5. 在下拉菜单中选择你想启动的程序（例如 Chrome、VS Code）。
6. 保存（取一个名字，比如“打开 Chrome”）。

接下来绑定快捷键：

1. 打开 **系统设置 → 键盘 → 快捷键**。
2. 左侧找到 **服务**（或 快速操作）。
3. 找到刚才保存的那个快速操作（比如“打开 Chrome”）。
4. 给它设置一个全局快捷键（例如 ⌘ + ⌥ + C）。

以后你就能用这个快捷键直接启动程序了。
