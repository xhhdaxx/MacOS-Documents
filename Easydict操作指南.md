# Easydict 常用操作指南

## ⌨️ 基本操作

### 1️⃣ 输入翻译

- **快捷键**：`⌥ + A`
- 弹出输入框 → 输入文本 → `Enter` 翻译

------

### 2️⃣ 划词翻译（鼠标）

- 鼠标选中文字
- 文字旁会出现小图标
- 鼠标悬停即可显示翻译结果

------

### 3️⃣ 划词翻译（快捷键）

- **快捷键**：`⌥ + D`
- 选中文字后按快捷键触发翻译

------

### 4️⃣ 截图 OCR 翻译

- **快捷键**：`⌥ + S`
- 框选屏幕区域
- 自动识别图片文字并翻译

------

### 5️⃣ 静默截图 OCR（只复制文本）

- **快捷键**：`⌥ + ⇧ + S`
- 框选区域后：
  - 不显示翻译窗口
  - OCR 文本直接复制到剪贴板

------

## 🔊 发音相关

### 自动发音

- 查询单词后可自动播放发音（可在设置中开启/关闭）

### 手动播放发音

- **快捷键**：`⌘ + S`

------

## 🌍 翻译方向与语言

- 自动识别源语言
- 可在设置中指定 **目标语言**
- 支持多语言自动切换

------

## 🔀 多翻译结果

- 同时显示多个翻译/词典结果
- 可在设置中：
  - 启用 / 禁用某个服务
  - 调整显示顺序

------

## ⚙️ 快捷键说明（可自定义）

| 功能     | 默认快捷键  |
| -------- | ----------- |
| 输入翻译 | `⌥ + A`     |
| 划词翻译 | `⌥ + D`     |
| 截图翻译 | `⌥ + S`     |
| 静默 OCR | `⌥ + ⇧ + S` |
| 播放发音 | `⌘ + S`     |

> 所有快捷键均可在 **偏好设置** 中修改

------

## 🧠 使用建议（高频）

- **读英文论文 / 文档**：划词翻译
- **看 PDF / 图片内容**：截图 OCR
- **写作 / 翻译整段文字**：输入翻译
- **只要文字不要弹窗**：静默 OCR
