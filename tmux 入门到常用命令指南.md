#  tmux 入门到常用命令指南

## 1️⃣ tmux 基础概念

- **tmux** 是终端复用工具，可以在一个终端中创建多个“会话（session）”、分屏窗口（window）、面板（pane）。

- 核心层级：

  ```
  会话(Session) → 窗口(Window) → 面板(Pane)
  ```

- 默认前缀键：`Ctrl + b`（先按 Ctrl+b，然后再按后续命令键）

------

## 2️⃣ 会话管理

| 命令                           | 功能                       |
| ------------------------------ | -------------------------- |
| `tmux` 或 `tmux new -s <name>` | 创建新会话（可指定名称）   |
| `tmux ls`                      | 列出当前所有会话           |
| `tmux attach -t <name>`        | 连接已有会话               |
| `tmux detach` (`Ctrl+b d`)     | 暂时离开当前会话，后台运行 |
| `tmux kill-session -t <name>`  | 关闭指定会话               |

💡 Tip：如果你经常断线或者在远程服务器上操作，tmux 可以让程序持续运行，非常方便。

------

## 3️⃣ 窗口（Window）管理

| 命令                    | 功能                        |
| ----------------------- | --------------------------- |
| `Ctrl+b c`              | 创建新窗口                  |
| `Ctrl+b ,`              | 重命名当前窗口              |
| `Ctrl+b w`              | 显示窗口列表，选择切换      |
| `Ctrl+b n` / `Ctrl+b p` | 切换到下一个/上一个窗口     |
| `Ctrl+b <数字>`         | 切换到指定窗口编号（0开始） |
| `Ctrl+b &`              | 关闭当前窗口                |

------

## 4️⃣ 面板（Pane）管理

| 命令          | 功能                         |
| ------------- | ---------------------------- |
| `Ctrl+b "`    | 垂直分割当前窗口（上下两块） |
| `Ctrl+b %`    | 水平分割当前窗口（左右两块） |
| `Ctrl+b o`    | 切换到下一个面板             |
| `Ctrl+b q`    | 显示面板编号（短时间）       |
| `Ctrl+b x`    | 关闭当前面板                 |
| `Ctrl+b ↑↓←→` | 在面板之间移动               |
| `Ctrl+b z`    | 放大/恢复当前面板            |

💡 Tip：多面板可以同时运行不同的命令，非常适合调试或监控多个进程。

------

## 5️⃣ 复制模式（Copy Mode）

- 进入复制模式：`Ctrl+b [`
- 移动光标：`↑↓←→`
- 搜索文本：`/` + 关键词
- 选中并复制：按空格开始选择 → 移动到结束 → 回车复制
- 粘贴：`Ctrl+b ]`

------

## 6️⃣ 其他实用命令

| 命令                            | 功能               |
| ------------------------------- | ------------------ |
| `Ctrl+b t`                      | 显示当前时间       |
| `Ctrl+b ?`                      | 显示所有快捷键列表 |
| `Ctrl+b s`                      | 切换会话           |
| `tmux source-file ~/.tmux.conf` | 重新加载配置文件   |

------

## 7️⃣ 使用示例

假设你在远程服务器上，需要同时运行日志监控和脚本：

```bash
# 1. 创建会话
tmux new -s mysession

# 2. 分屏
Ctrl+b %       # 左右分屏
Ctrl+b "       # 上下分屏

# 3. 在不同面板运行不同命令
# 左上 panel：python script.py
# 右 panel：tail -f log.txt

# 4. 暂时离开会话
Ctrl+b d

# 5. 后续重新连接
tmux attach -t mysession
```

这样即使你退出 SSH，程序仍然在后台运行。
